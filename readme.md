# ngKarma

---

A simple Pinterest-style Reddit user interface.

---

** Note: This project is in a functional but lackluster state. Check it out at [ngKarma.com](http://ngkarma.com).

### Requirements

* Laravel
* Bower
* Composer
* AngularJS
* MasonryJS
* jQuery
* MongoDB
